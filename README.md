# linebot-template

A line bot template with simply echo function

## Features

- Using
  - [FastAPI](https://fastapi.tiangolo.com/)
  - [Line Bot SDK Python](https://github.com/line/line-bot-sdk-python)
  - VSCode devcontainer
  
- Support
  - Heroku
  - Docker

- pre-installed
  - git
  - heroku cli
